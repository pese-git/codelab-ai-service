## Инструкция по заполнению шаблона

1. **Заполняйте по ходу эксперимента** - не оставляйте пустые поля до конца
2. **Используйте конкретные числа** - избегайте формулировок типа "улучшилось"
3. **Добавляйте контекст** - объясняйте почему результаты именно такие
4. **Будьте объективны** - опирайтесь на данные, а не на ожидания
5. **Фиксируйте insights** - отмечайте неожиданные открытия
6. **Обосновывайте решения** - каждое заключение должно подкрепляться фактами

## Полезные формулы для расчетов

```
Task Success Rate = (successful_tasks / total_tasks) * 100%
Avg Time = sum(all_durations) / total_tasks
Avg Cost = sum(all_costs) / total_tasks
Effect Size (Cohen's d) = (mean_multi - mean_single) / pooled_std
```

## Критерии оценки

- **ПРОДОЛЖАТЬ**: TSR улучшение ≥15% И статистическая значимость
- **ДОРАБОТАТЬ**: TSR улучшение 5-15% ИЛИ неоднозначные результаты
- **ОСТАНОВИТЬ**: TSR улучшение <5% ИЛИ ухудшение показателей
